萌新入坑 7 天指南

必读
💡 核心思路：Heartopia 没有强制打卡，但前7天的选择会极大影响后续体验。跟着主线走→解锁爱好→建立赚钱循环，就是最优路径。
1 第1-2天：推主线 + 领奖励
  • 📦

    领取所有开服/预约奖励

    打开邮箱 → 领取月光水晶(400)、展览券(60)、愿望星(20)、金币(4000)。这些是前期最重要的资源。

  • 📖

    全力推进主线任务

    主线解锁约90%的核心功能——包括钓鱼、园艺、制作、建造平板。不做主线 = 锁死大部分内容。

  • 🛏️

    优先放置基础家具

    床（恢复体力）、工作台（制作）、烤箱（烹饪）、浴缸 — 这四件是前期最重要的家具。

  • 🚗

    领取免费载具

    前期会送小摩托，开服奖励还有白色轿车。走路虽浪漫，但地图巨大，有车效率翻倍。

2 第3-5天：解锁爱好 + 建立收入
  • 🎣

    完成钓鱼+园艺教程（强制解锁）

    钓鱼导师 Vanya、园艺导师 Blanc 会带你入门。这两项是赚钱的基础。

  • 🍳

    DG等级达标后，第一张爱好券解锁烹饪

    烹饪是最关键的爱好——把种的菜、钓的鱼做成菜肴，售价翻倍，还能高效回体力。

  • 🌾

    种田 → 收菜 → 做成菜 → 卖掉 = 第一桶金

    不要卖原材料！苹果做成苹果酱、鱼做成烤鱼，利润差3-5倍。

3 第6-7天：稳定日常 + 扩展玩法
  • 📋

    每天完成居民委托（DG等级关键来源）

    居民会要你拍照、交材料、钓鱼。完成委托 = DG经验 + 贡献徽章，徽章能换家具和服装。

  • 🦋

    DG等级6：解锁第二个爱好 — 推荐捕虫

    捕虫门槛低、收集乐趣大，且能和烹饪联动（部分菜品需要虫类材料）。

  • 🏠

    每天逛商店（家具+服装每日刷新）

    限量家具和稀有服装可能只出现一次，看到喜欢的果断拿下。

赚钱攻略 & 收益对比

⚠️ 新手最常见的错误：把原材料直接卖掉。永远记住——先加工，再出售。苹果 → 苹果酱，鱼 → 烤鱼，利润差3-5倍。
📊 赚钱方法收益对比
方法 效率评级 门槛 说明
🌾 种田 + 卖菜 ⭐⭐⭐⭐⭐ 极低 最稳定收入源。作物品质随种植次数提升，不需要肥料也能出高品质。推荐优先种高价值作物。
🍳 烹饪 + 卖菜 ⭐⭐⭐⭐ 需解锁烹饪 种田+烹饪联动 = 最佳赚钱循环。高品质菜肴售价可观，低品质自己吃回体力。
🎣 钓鱼出售 ⭐⭐⭐ 教程解锁 高星级鱼直接卖，低星级做成料理再卖。注意稀有鱼有时间+天气限制。
📋 每日/每周任务 ⭐⭐⭐ 无 稳定贡献徽章+金币，性价比最高的固定收入。
🫧 戳地图泡泡 ⭐⭐ 无 随机出金币、材料、服装、蓝图。路过顺手戳，不专门找。
🔄 赚钱闭环（推荐路线）
种田 → 收菜 → 烹饪加工 → 高品质 → 卖掉 → 低品质 → 回体力 → 继续种田

常见问题速查

Heartopia 的鱼类分布在河流、湖泊、海洋三种水域，稀有鱼还需要满足特定天气 + 时间段条件。 彩虹天气（同时满足晴天和雨天刷新池）是最佳钓鱼时机。具体每只鱼的条件请在解锁钓鱼后查看游戏内图鉴。

加好友：通过UID搜索添加，花费1个愿望星发送好友请求。
共建模式：DG等级到达8级后解锁，可以邀请一位好友在你的家园地块上实时共建。
私人小镇：花费月光水晶/钻石之心创建，只有你和好友能进入。

DG（发展家等级）是游戏最核心的进度系统。提升方式：
① 每天完成居民委托——主要经验来源，尽量全部做完
② 每日/每周任务——奖励贡献徽章
③ 主线任务——前期一次性大量经验
DG等级解锁：新爱好、新家具、背包扩容、蓝图(Lv.8)、共建(Lv.8)等。

工具设计为可修复的，不需要重新购买。使用维修套件（Repair Kit）即可修复。 维修套件可以通过签到奖励、任务奖励获得，也可以在商店购买。每件工具都有自己的耐久值，耐久归零前修好就行。

DG等级8解锁蓝图功能。蓝图允许你：
① 保存多套房屋设计方案，随时切换
② 在不同地块上建造，每个地块独立保存
③ 购买社区蓝图直接套用（不用自己设计）
这是建造党的核心功能，建议尽早冲到DG 8。
